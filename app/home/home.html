This is version of the DigitalEdge Visualization Template.

Build analytic apps easily!

This template provides a code base from which advanced analytics web applications can be built. It is built using AngularJS and contains recommended coding conventions and styles. The template contains samples of different angular components so that bootstrapping your project is as easy as copying and modifying the existing files. In addition, most of the required development and testing tools are pre-configured.
